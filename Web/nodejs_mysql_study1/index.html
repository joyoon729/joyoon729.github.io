MySQL package 설치

Node.js 에는 기본 제공하는 MySQL 모듈이 없습니다.

때문에 npm 을 이용해 MySQL 패키지를 따로 설치하였습니다.


npm install 을 이용하면 package.json 에 있는 의존성들(dependencies)을 자동으로 검사해 필요한 패키지들을 알아서 설치해줍니다.

여기서 --save 옵션은 package.json 의 dependencies 에 mysql 을 추가해주는 옵션입니다.

안정성을 위해 옵션을 추가해주는 것이 좋습니다.
